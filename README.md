# Youssef Hammouch Portfolio Website v2

Static premium portfolio website with:
- anonymized enterprise-style dashboard gallery
- story-driven consulting case studies
- banking case study included
- responsive layout ready for static hosting

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Run locally
Open `index.html` in any browser.

## Deploy
Compatible with:
- Netlify
- Vercel static hosting
- GitHub Pages
- Any shared hosting

## Notes
- Company names are intentionally hidden.
- The dashboards are presentation-ready portfolio visuals.
- Case studies are anonymized representative project stories.
